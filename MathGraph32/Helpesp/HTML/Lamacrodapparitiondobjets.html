Macro unmasking objects.


An macro unmasking objects is created through menu Macros - New macro - Unmasking objects.

It's purpose is to make visible hidden objects.

Clicking on the macro button will make these objects become visible.


Two possibilities :

  • If radio button Click for next object is selected, you must click as many times as the macro manages objects.
  • If radio button Click for next object is not selected, a single click all the objects appear.
