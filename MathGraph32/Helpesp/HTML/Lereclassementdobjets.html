Object reclassification

In certain cases, you will need to change the order objects were created in.


Here is an example :

In the back color, you create two free points O and O then the circle center O through B point.

The create the line the line joining points O and A.

You activate red color, and choose the full filling style in the palette (use the scrollbar).

Click the on tool . Then click on the circle to get it filled.

The surface created hides a part of line AB.

Then use menu Reclassification towards list end - of a graphical object and click on line AB.

The part of the line which was hidden is now visible because the line has been reclassified as far as possible in the list of created objects.

If you use the protocol tool (shortcut F5) you will notice that the line is now the last of objects created.


In a similar way it is possible to reclassify a graphical object or a numerical object as far as possible either towards the end of the list or towards the start of the list.

To modify or reclassify a numerical object you can use icon (keyboard shortcut F6).


A macro can be reclassified either by clicking on it's title or through menu Macro - Macro modification or reclassification.
