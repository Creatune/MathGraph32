Figure options

Options of the currently edited figure are changed via menu Options - Current figure.

A dialog box pops up.


  • Unity angle tab allows the choice of the angle unity : radian or degree.
  • Background Color tab allows changing the background color of the current figure.
  • Default animation macro allows the choice of an animation macro through which the figure will be continuously animated when displayed by MathGraph32 applet.
  • Background image allows choosing an image file for he background of the image. This image file must lay in the same directory as the figure or in a sub-directory.
  • Fixed objects tab is for allowing or not angle marks or displays to me moveable through tool (for translation of the figure).
  • Formula editors tells if button values or predefined functions will be available when clicking on a formula editor.
  • Menu Customisation is for allowing only the use of chosen objects in this figure. This will allow teachers to make an exercise using only special tools.
  • Starting macro tab allows you to select a starting macro that will be executed once the figure loaded. This macro can only be a macro of iterative construction, a macro of recursive construction or a macro executing a sequence of macros of one of the two preceeding types.
