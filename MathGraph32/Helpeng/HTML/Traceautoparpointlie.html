Auto trace through linked point macro

Macros ›› Different kind of macros ››

Such a macro is created through menu Macros - New macro - Automatic trace through linked point.


When activated, this macro leaves on the figure the trace of a list of objects when a linked point is moving on the object it is linked to.


To be noticed : The trace is not permanent and will disappear with tool .
