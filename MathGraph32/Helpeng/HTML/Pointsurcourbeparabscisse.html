Point on curve through abscissa

Curves and graphs ››

This is not a real object but the result of an internal construction.


For this tool to be used, your figure must contain a frame.


Use toll or menu item Create - Point on function curve - through abscissa.

A dialog box pops up.

Click on the function and the frame.

Specify the abscissa of the point to be created. You may refer to an existent value or enter a calculus. In the second case a calculus will be added to the list of objects created the name of which will start with abs.

Enter the name for the calculus that will contain the image of the abscissa through the function.

A point appears. This point is on the function curve.

It is a final object of a construction named Point on curve through abscissa.
