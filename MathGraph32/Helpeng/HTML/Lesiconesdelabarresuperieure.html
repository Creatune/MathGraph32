New calculus and drawing of the whole figure, including new values pour rand() function.

Translation of figure. Click and drag the mouse.

Capture of an object : free point, linked point, angle mark, value display, text display, macro title.

Rename a point or a line. Simply click on the point or the line. If a name of a point or line is near the mouse pointer, you can drag it.

Changing the color or drawing style of an object (line style, point style, arrow style, filling style).

Modify a graphical object when this object needs a dialog box to be changed (keyboard shortcut F7). For example, clicking on a point image of another through a rotation will get a popup dialog box appear allowing the change of angle.

Simply click on the object to be modified.

To list, modify or  reclassify numerical objects (keyboard shortcut F6).

Deletion of a graphical object (keyboard shortcut Del). The object will be delated as well as dependant objects.

Deletion of a numerical object (keyboard shortcut Del). The object will be delated as well as dependant objects. A dialog box pop up.

Masking an object (the object still exists but is no longer visible).

Unmasking a hidden object. Most of hidden objects are in dotted style for this tool.

Launches the protocol display of the figure. (keyboard shortcut F5).

To get information on the relation between an point and an object or between two objets (line, half-line, segment, circle or circle arc).

To display the text of the figure.

Execution of a macro when clicking on the title.

Activation or deactivation of mode Trace. When mode Trace is on, traces you get via menu Move - Objects trace or via a macro generating  a trace remain permanently as long as mode Trace is on and the window containing the figure is not resized.
