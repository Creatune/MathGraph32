Removing objects from a macro

Macros ››

Some macros manage a list of objects : unmasking objects macros, masking objects macros, animation with trace macros, automatic trace macros, animation with loop macros, animation with trace macros.


If you want to remove objects from the list of objects managed by a macro :

  • Use menu Macros - remove objects from macro.
  • Click on the macro objects are to  be added to. Objects managed by the macro blink.
  • Click on the blinking objects you want to be removed from the macro.
  • Right click to end.


At least one object must remain in the macro list.
