Visual help for objects creation

Software presentation ››

MathGraph32 gives graphic information on a graphic object while creating the object.

For example, when using the icon of circle creation through two points, after clicking on the center, a temporary circle appears, which center is the first point clicked  and including a  temporary point located at the mouse pointer position.


You can activate or disable visual help creation through menu Options - Visual help activated  (shortcut F2).
