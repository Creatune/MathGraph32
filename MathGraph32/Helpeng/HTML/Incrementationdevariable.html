Variable increment macro

Macros ›› Different kind of macros ››

A macro incrementing a variable is created through menu Macros - New macro - Variable increment.

When activated, such a macro adds the step of the variable to the current value (if the new value doesn't exceed the maxi value) and puts the result in the current value.
