Links between objects

Link point-object.


A linked point is a point linked to an object. It can me moved but must stay on the object.

Turning a linked point in a free point.

The link between a point and an object can be deleted via menu Modify - Link suppression - Between point and object.

Turning a free point in a linked point.

A free point can be turned in a linked point via menu Modify - Link suppression - between point and object.

Click first on the free point which is to become a linked point. Then click on the object the point must be linked to. The link creation is impossible if the object the free point is to be linked to depends on the free point.


Link display-point.


Text displays, value displays, macros can be linked to a point.

A text display linked to a point is created via menu Create - Text display - Linked to a point (shortcut Ctrl + K).

A value display linked to a point is created via Create - Value display - Linked to a point (shortcut Ctrl + L).


Turning a linked display in a free display.

The link between a display and an object can be deleted via menu Modify - Link suppression - Between display and point.

Turning a free display in a linked display.

A free display can be turned in a display linked to a point. Click first on the free display. Click the on the point the display is to be linked to.
