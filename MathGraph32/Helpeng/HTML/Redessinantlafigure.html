Updating figure macro

Macros ›› Different kind of macros ››

Such a macro is created through menu Macros - New macro - Blinking of objects.


When activated such a macro erases the display and redraws the figure. It is useful to erase traces produced by another macro.


To be noticed :

  • Only one macro of this type can be created in a figure.
  • If the figure uses calls of rand() function, activating this macro will cause a new value to be affected for all the calls of random function, thus allowing a random experience to be updated.
