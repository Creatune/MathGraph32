Preferences

You can personalize MathGraph32 use through menu Options - Preferences.


Angle unity tab changes the angle unity of the future figures to be created.


Language tab allows to change the language used by MathGraph32 (French, Spanish or English).

Level tab is for choosing the level of use. For example, calculus on complex numbers will only be available in level Advanced level with complex number calculus functionalities. In elementary or mid-level  level, length measures and angle measures will be automatically displayed on the figure.


Toolbar tab allows to change  the size of the icons of the left toolbar and also to get the icons of the same type surrounded with the same color.


Figure translation tab specifies if free displays have to be or not moved while translating all the figure.


Animation tab specify the way to animate the figure using menu Move - Linked point animation.


Files directory tab allows the choice of the default directory when opening or saving a file.


Constructions directory tab allows the choice of the default directory when opening or saving a construction on file.


Clipboard tab allow to give a scale factor by wich the two dimensions of the image will be multiplied when copying the figure in the clipboard.


To get a personalized level of use  :


Warning : If you want to get a level personalized you must have administrator rights on the computer.

Use menu Options - Personalization of level - Level (1 to 4) to personalize the chosen level.

In the field Name of level, choose the description for your personalized level.

In the tree of menu items, select the tools that will be available (or not) for this level.

When the tools are selected, click on the button Save.

On the next start of MathGraph32, for the chosen level, the chosen tools will be available if you check the box Personalized levels in the menu Option - Preferences, tab Personalized levels.


To get MathGraph32 levels personalized from a machine of reference :


Personalized levels are saved in four files named  level1.mgj, level2.mgj, level3.mgj and level4.mgj (from the lowest level to the highest one).


To get the same personalized levels in use on another computer, find the directory containing the executable of MathGraph32 (for the Windows installed version, it is in the Program Files/MathGraph32Java directory,  file mathgraph32Java.exe).

Copy the four files named  level1.mgj, level2.mgj, level3.mgj and level4.mgj in the same directory on the destination computer.

On the destination computer, on the next start of MathGraph32, use Menu Option Preferences, tab Personalization of levels and check the checkbox Personalized level.

On the next start of MathGraph32, the personalized levels will be in use.
