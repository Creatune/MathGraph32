Adding objects to a macro

Macros ››

Some macros manage a list of objects : unmasking objects macros, masking objects macros, animation with trace macros, automatic trace macros, animation with loop macros, animation with trace macros.


If you want to add objects to the list of objects managed by a macro :

Use menu Macros - add objects to macro.

A dialog box pops up.

  • Click on the macro you wan objects added to.
  • Click on the objects to be added to the macro.
  • Right click to end.
