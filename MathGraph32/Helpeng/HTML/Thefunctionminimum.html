The function minimum

Numerical objects ›› Numerical objects of calculation type ››

Allows the user to calculate the abscissa of the minimum of a function on an interval [a; b].

The value is ascertained to be accurate only if the function is decreasing from a up to the maximum and then increasing until b.


In the dialog box popping up you must :

  • Select the function in the list of functions already created.
  • Choose the a value.
  • Choose the b value.
  • Choose the uncertainty value.
