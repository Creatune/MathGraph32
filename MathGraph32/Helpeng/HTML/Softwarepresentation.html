Images

Since version 4.4, images are directly included in the generated mgj files.


You can create a free image display by clicking on icon .

You can  create an image display linked to a point by clicing on icon .


Then you will be asked to fetch the image on your computer.



A MathGraph32 figure can also contain a background image. This image is alos directly included in the mgj file generated.


You can choose the background image with menu item Options - Background image choice.


You can delete the background image with menu item Delete background image.


The background image can be momentarily hidden by unchecking the Background image visible checkbox in the Options menu.
