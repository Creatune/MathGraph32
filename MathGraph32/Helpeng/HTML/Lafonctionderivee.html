The derivative function

Numerical objects ›› Numerical objects of calculation type ››

MathGraph32 is able to calculate the formal derivative function of a real function.


It is created via menu Calculus - New real calculus - Derivate function.


A dialog box appears displaying all available functions (except function derivative function of which is already defined).


Select the function and specify a name for the derivative.


To be noted : It is the result of a formal calculus but the formula of the derivative is not available to the user.
