Point on curve through linked point

Curves and graphs ››

This is not a real object but the result of an internal construction.


For this tool to be used, your figure must contain a frame and a linked point (linked the the x-axis for example).

Use menu item Create - Point on function curve - through linked point.

A dialog box pops up.


Click on the function and the frame.

Enter a name for the abscissa of the linked point in the frame.

Enter a name for the image of the abscissa through the function.

Click OK.

Then click on the linked point to finish.

A new point appears. This point is a final object of a construction named Point on curve through linked point.

In the final objects you will also get :

a calculus containing the abscissa of the linked point clicked.

a calculus containing the image of the abscissa through the function.
