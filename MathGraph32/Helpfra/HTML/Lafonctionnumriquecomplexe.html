La fonction numérique complexe

Les objets numériques ›› Les objets numériques de type calcul ››

C'est une fonction numérique complexe d'une variable complexe.

On crée une fonction numérique complexe avec le menu Calculs - Nouveau calcul dans C - Nouvelle fonction complexe.

On peut modifier une fonction avec le menu Modifier -Formule (raccourci clavier Ctrl + i).

Par exemple, pour définir une fonction par f(z) = (z + 1/z)2, on choisit comme nom f et peut choisir comme variable formelle z et comme formule
(z + 1/z)^2 mais aussi choisir comme vatiable formelle t et entrer comme formule (t + 1/t)^2.
