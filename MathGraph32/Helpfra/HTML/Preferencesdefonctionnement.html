Préférences de fonctionnement

Vous pouvez personnaliser le fonctionnement de MathGraph32 à l'aide du menu Options - Préférences.


L'onglet Unité d'angle vous permet de choisir l'unité d'angle pour les futures figures créées.


L'onglet Langue vous permet de choisir la langue utilisée par le logiciel (français, anglais et espagnol).


L'onglet Barre d'outils vous permet de choisir la taille (petite ou grande) des icônes de la barre d'outils de gauche et d'entourer ou non dans une même couleur les icônes ayant des fonctions similaires.


L'onglet Niveau vous permet de choisir le niveau des outils proposés. Par exemple, le calcul sur les nombres complexes ne sera disponible que dans le niveau Avancé avec prise en charge des complexes.

L'onglet Translation figure vous permet de choisir si, lors de l'outil de translation de la figure, les affichages libres (c'est à dire non liés à des points) seront dépacés ou non.


L'onglet Animation vous permet de chosir comment sera animée la figure quen vous utilsiez le menu Mouvoir - Animer un point lié.


L'onglet Répertoire fichiers vous permet de spécifier le répertoire par défaut quand vous ouvrez ou enregistrez une figure.


L'onglet Répertoire constructions vous permet de spécifier le répertoire par défaut quand vous implémentez une construction depuis un fichier ou enregistrez une figure.


L'onglet Personnalisation des niveaux permet de personnaliser les quatre niveaux de fonctionnement de MathGraph32 :


Vouz pouvez, pour chaque niveau du plus faible au plus élevé, choisir si les mesures de longueurs et d'angles non orientés doivent être affichés ou non sur la figure en cochant la case correspondante (pour les niveaux par défaut, dans les niveaux élémentaire et collège, les mesures d'angles et de longueurs sont directement affichées sur la figure).

Les mesures d'angle ne sont affichées sur la figure qui si celle-ci utilise le mode degré.

Mais il est aussi possible de changer les outils permis pour chaque niveau.

Pour utiliser les niveaux personnalisés, cochez la case Niveau personnalisé.

La version Française est fournie avec quatre niveux personnalisés (que vous pouvez modifier) : Ecole élémentaire, sixième, cinquième et quatrième - troisième.

Une fois la case Niveau personnalisé cochée, rien ne se passera si vous n'avez pas utilisé le menu Options - Personnalisation du niveau - Niveau 1 à 4 pour personnaliser les niveaux souhaités.


L'onglet Presse-papier permet de spécifier par quel coefficient les deux dimensions de l'image en cm telle qu'elle apparaît sur l'écran doivent être modifiées quand elle est collée dans le presse-papier.


Pour personnaliser un niveau de fonctionnement :


Attention : Pour personnaliser un niveau de fonctionnement, vous devrez souvent posséder les droits administrateurs sur la machine.

Utilisez le menu Options - Personnalisation du niveau - Niveau (1 à 4) pour personnaliser le niveau choisi.

Dans le champ Nom du niveau, entrez d'abord le descripteur de votre choix pour le niveau.

Dans l'arbre des menus, choisissez les outils qui seront autorisés ou non à l'utilisateur dans ce niveau.

Une fois les outils sélectionnés (il peut s'agir des outils permis ou interdits), cliquez sur le bouton Sauvegarder.

Au prochain redémarrage de MathGraph32, pour le niveau choisi, ce seront les outils disposibles pour ce niveau qui seront accessibles à condition d'avoir coché la case Niveaux personnalisés dans le menu Options - Préférences, onglet Personnalisation des niveaux.


Pour personnaliser les niveaux de fonctionnement de plusieurs machines de façon rapide :


Les niveaux personnalisés sont sauvegardés dans quatre fichiers nommés level1.mgj, level2.mgj, level3.mgj et level4.mgj (du niveau le plus faible au niveau le plus élevé).


Pour utiliser les mêmes niveaux personnalisés sur d'autres machines, trouvez l'emplacement dans lequel figure le jar exécutable de MathGraph32 (pour la version installée sous Windows, il s'agit du dossier nommé MathGraph32 Java dans Program Files). Ce jar exécutable se nomme MathGraph32.jar (MathGraph32.exe pour la version installée sous Windows).

Copiez les quatre fichiers nommés level1.mgj, level2.mgj, level3.mgj et level4.mgj et collez les dans le même dossier sur la machine de destination.

Attention : il sera peut-être nécessaire d'avoir les droits administrateurs sur la machine de destination pour faire cette opération.

Sur les machines de destination, il faudra, lors de la prochaine utilisation de MathGraph32, utiliser le menu Option - Préférences, onglet Personnalisation des niveaux et cocher la case Niveau personnalisé. Ensuite, à chaque démarrage, les machines utiliseront les niveaux personnalisés.
