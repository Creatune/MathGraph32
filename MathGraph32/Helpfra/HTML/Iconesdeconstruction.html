Pour créer l'image d'objets par une translation. Cliquez d'abord sur l'origine puis l'extrémité du vecteur de la translation. Le vecteur de la translation clignote. Cliquez ensuite sur un objet pour obtenir son image (droite, demi-droite, segment, cercle, arc de cercle, polygone, ligne brisée). Tant que le vecteur clignote il suffit de cliquer sur un objet pour avoir son image.

Pour créer l'image d'objets par une translation dont on donne les coordonnées du vecteur dans un repère. Choisissez d'abord les coordonnées du vecteur et le repère dans la boîte de dialogue puis cliquez sur un objet pour obtenir son image  (droite, demi-droite, segment, cercle, arc de cercle, polygone, ligne brisée). Tant que le vecteur clignote il suffit de cliquer sur un objet pour avoir son image.

Pour créer l'image d'objets par une symétrie axiale. Cliquez d'abord sur l'axe de symétrie qui clignote (droite, demi-droite, segment ou vecteur). Cliquez ensuite sur un objet pour obtenir son image (droite, demi-droite, segment, cercle, arc de cercle, polygone, ligne brisée).Tant que l'axe de symétrie clignote il suffit de cliquer sur un objet pour avoir son image.

Pour créer l'image d'objets par une symétrie centrale. Cliquez d'abord sur  le centre de symétrie qui clignote. Cliquez ensuite sur un objet pour obtenir son image (droite, demi-droite, segment, cercle, arc de cercle, polygone, ligne brisée ).Tant que le centre de symétrie clignote il suffit de cliquer sur un objet pour avoir son image.

Pour créer l'image d'objets par une rotation. Cliquez d'abord sur  le centre de symétrie qui clignote. Une boîte de dialogue s'ouvre pour vous demander l'angle de la rotation.Cliquez ensuite sur un objet pour obtenir son image (droite, demi-droite, segment, cercle, arc de cercle, polygone, ligne brisée ).Tant que le centre de la rotation clignote il suffit de cliquer sur un objet pour avoir son image.

Pour créer l'image d'objets par une homothétie. Cliquez d'abord sur  le centre de l'homothétie qui clignote. Une boîte de dialogue s'ouvre pour vous demander le rapport de l'homothétie.Cliquez ensuite sur un objet pour obtenir son image (droite, demi-droite, segment, cercle, arc de cercle, polygone, ligne brisée ).Tant que le centre de l'homothétie clignote il suffit de cliquer sur un objet pour avoir son image.

Pour créer l'image d'objets par une similitude directe. Cliquez d'abord sur  le centre de la similitude qui clignote. Une boîte de dialogue s'ouvre pour vous demander le rapport et l'angle de la similitude.Cliquez ensuite sur un objet pour obtenir son image (droite, demi-droite, segment, cercle, arc de cercle, polygone, ligne brisée ).Tant que le centre de l'homothétie clignote il suffit de cliquer sur un objet pour avoir son image.
