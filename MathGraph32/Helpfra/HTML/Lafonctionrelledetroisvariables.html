La fonction réelle de trois variables

Les objets numériques ›› Les objets numériques de type calcul ››

C'est une fonction numérique réelle de trois variables réelles.

On crée une fonction numérique réelle de trois variables avec le menu Calculs - Nouveau calcul dans R - Fonction réelle de trois variables.

On peut modifier une fonction avec l'icône .

Par exemple, on peut définir une fonction réelle de trois variables par f(x, y, z) = 2x²yz+ 3x + z + 1. Pour cela on entre f comme nom de la fonction,

x, y et z comme nom pour les trois variables et comme formule 2*x^2*y*z+3*x+z+1.
