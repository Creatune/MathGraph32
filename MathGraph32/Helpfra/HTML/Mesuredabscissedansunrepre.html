Mesure d'abscisse dans un repère

Les objets numériques ›› Les mesures ››

Pour mesurer l'abscisse d'un point dans un repère, utilisez l'icône .


Cliquez simplement sur le point dont vous voulez mesurer l'abscisse.

Si vous avez créé plusieurs repères, une boîte de dialogue vous demandera d'abord de choisir le repère.


Le point mesuré doit être nommé.

S'il ne l'est pas, un nom lui sera attribué.


Si vous avez mesuré l'abscisse de M dans le repère (O, I, J) elle est notée xCoord(M,O,I,J).
