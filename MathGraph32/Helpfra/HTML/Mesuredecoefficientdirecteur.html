Mesure de coefficient directeur

Les objets numériques ›› Les mesures ››

Pour mesurer le coefficient directeur d'une droite dans un repère, utilisez l'cône  ou le menu Calculs - Mesurer - Coefficient directeur de droite.


Votre figure doit être munie d'au moins un repère.


Cliquez d'abord sur la droite.

Une boîte de dialogue s'ouvre.

Entrez dans cette boîte le nom choisi pour la mesure et cliquez sur le repère dans laquelle cette mesure doit être effectuée.
