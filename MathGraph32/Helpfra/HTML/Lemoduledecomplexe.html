Le module de complexe

Les objets numériques ›› Les objets numériques de type calcul ››

Une formule de calcul réel ne pouvant pas faire référence à un nombre complexe, si vous voulez utiliser le module d'un nombre complexe dans un calcul réel, vous devez d'abord créer un calcul réel égal au module de ce complexe.

Vous pourrez alors faire référence à ce calcul (qui est un calcul considéré comme réel) dans un autre calcul réel.

Prenons un exemple :

Vous avez mesuré l'affixe d'un point M dans un repère (O;I,J). Cette affixe est notée Aff(M,O,I,J) par MathGraph32 et vous souhaitez créer un cercle dont le rayon soit la moitié de cette affixe.

Vous devez alors utiliser le menu Calculs - Nouveau calcul dans R - Module de complexe.

Dans la boîte de dialogue qui s'ouvre sélectionnez l'affixe notée Aff(M,O,I,J).

Dans le champ Nom du calcul, entrez a.

Validez par OK.

Quand vous créerez le cercle défini par son centre et son rayon vous pourrez alors entrer dans le champ Rayon a/2.
