Couleurs et transparence

Présentation du logiciel ››

Une palette de 14 couleurs de base est disponible à droite de la fenêtre du logiciel.

Sous cette palette figure un bouton ovale permettant un choix personnalisé pour la couleur active .

La couleur de ce bouton reflète la couleur active pour les nouveaux objets créés.

C'est aussi la couleur qui sera donnée à un objet graphique quand on clique sur celui-ci avec l'outil .


Depuis la version 2.0, MathGraph32 peut utiliser des effets de transparence pour le remplissage des surfaces.


Le coefficient d'opacité est déterminé en déplaçant le curseur figurant sous le bouton ovale de choix de couleur

Ce coefficient est mémorisé dans la figure quand on l'enregistre.

Un deuxième bouton montre quelle sera la réelle couleur affichée (sur un fond blanc).

Cet effet de transparence n'est utilisé que pour le choix de remplissage plein avec transparence dans la palette de style de remplissage.

Le choix de remplissage plein sans effet de transparence est symbolisé par .

Les autres sytyles de remplissage avec hachures n'utilisent pas d'effet de transparence.


Quand les figures créées avant la version 2.0 sont chargées, le coefficient d'opacité qui leur est attribué est de 1. Ainsi elles sont affichées comme dans les versions précédentes. Ce coefficient peut être modifié.
