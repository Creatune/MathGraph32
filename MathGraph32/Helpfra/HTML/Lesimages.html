Les images

Depuis la version 4.4, les images sont directement enregistrées dans le fichier mgj de la figure et les fichiers sources n'ont plus à être présents dans le même dossier que la figure (ou un sous-dossier).

On crée un affichage d'image libre avec l'icône et un affichage d'image lié à un point avec l'icône .

Il suffit ensuite d'aller chercher le fichier image dans un dossier de votre ordinateur.


Une figure peut aussi comprendre une image de fond. Celle-ci est aussi enregistrée dans le fichier mgj de la figure depuis la version 4.4.

On choisit une image de fond pour la figure avec le menu Options - Choix l'image de fond.

On peut supprimer l'image de fond de la figure avec le menu Options - Suppression de l'image de fond.


L'image de fond de la figure peut être temporairement cachée en décochant la case Image de fond visible dans le menu Options.
