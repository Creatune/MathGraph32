Trace de points et d'objets

Lorsque l'icône est activée, les points marqués pour la trace laissent un trace lors de leur déplacement.

Certaines macros ont aussi pour effet de laisser des traces pour certains objets quand elles sont exécutées. Lorsque le mode Trace est activé, ces traces restent visibles.



Pour marquer un point pour la trace, utiliser le menu Modifier - Marquer une point pour la trace (icône ).

Pour démarquer un point pour la trace, utiliser le menu Modifier - Démarquer une point pour la trace (icône ).



A noter : Le mode trace d'une figure est enregistré avec celle-ci et ce mode sera activé lorque la figure sera ouverte à nouveau.
