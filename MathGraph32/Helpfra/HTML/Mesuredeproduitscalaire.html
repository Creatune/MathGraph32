Mesure de produit scalaire

Les objets numériques ›› Les mesures ››

Pour mesurer le produit scalaire de deux vecteurs votre figure doit comporter une longueur unité.

Cliquez sur l'icône de mesure de produit scalaire.

Cliquez ensuite sur les deux vecteurs dont vous souhaitez mesurer le produit scalaire.


A noter : Les extrémités des deux vecteurs doivent être nommées. Si elles ne sont pas nommées, un nom leur sera attribué automatiquement.
