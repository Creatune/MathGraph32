Le test d'existence

Les objets numériques ›› Les objets numériques de type calcul ››

Un test d’existence est une valeur qui permet de savoir si une valeur (calcul, mesure ou fonction) déjà créée existe ou non.

La valeur renvoyée est 1 si la valeur associée existe et 0 sinon.

Imaginons par exemple que vous avez créé une figure munie d’une longueur unité et créé un point M d’intersection d’un cercle de centre O avec une droite. Pour certaines positions de la droite, il peut arriver que ce point M n’existe plus. Pour savoir si ce point existe ou non, mesurez d’abord la longueur OM (en effet on ne peut pas directement tester l’existence d’un point), puis créez un test d’existence de cette longueur à l’aide du menu Calculs - Calcul algébrique dans R - Nouveau test d’existence.
