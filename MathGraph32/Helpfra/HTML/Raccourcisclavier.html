Raccourcis clavier

F1

Lance l'aide de MathGraph32 si elle est disponible. Pour qu'elle le soit le fichier d'aide doit être présent dans le répertoire contenant le jar de MathGraph32. Ce fichier doit avoir pour nom MathGraph32Helplan.htm où lan représente la langue utilisée. Dans le cas où la langue utilisée est le Français, ce fichier est nommé MathGraph32fra.htm.

F2

Activation ou désactivation de l'aide visuelle à la cration d'objets.

F3

Activation ou désactivation des noms automatiques.

F4

Implémentation d'une construction prédéfinie.

F5

Lance l'historique de la figure (boîte de dialogue flottante).

F6

Affiche dans une boîte de dialogue tous les objets numériques existants,avec possibilité de les supprimer ou reclasser.

F7

Activtion de l'outil de modification d'objet graphique.

F8

Active l'outil d'exécution d'une macro (icône ).

F9

Active l'outil précédemment employé dans le cas où le dernier outil utilisé avait une action unique (par exemple un outil de mesure).

F10

Affiche des informations sur une construction en cours d'implémentation, sinon active le menu de la figure.

F11

Implémentation d'une construction contenue dans la figure.

F12

Active l'outil qui, quand on clique sur un obet graphique, affiche des informations (et un numéro pour communication en JavaScript)

Suppr

Active l'outil de suppression d'objet graphique ().

Ctrl + A

Mesure de l'affixe complexe d'un point dans un repère (icône ).

Ctrl + B

Création d'un point défini par son affixe complexe d'un point dans un repère.

Ctrl + C

Copier la figure dans le presse-papier.

Ctrl + D

Création d'une droite par son coefficient directeur (icône ).

Ctrl + E

Création d'un calcul réel

Ctrl + Maj + E

Création d'un calcul complexe

Ctrl + F

Création d'une fonction numérique d'une variable réelle.

Ctrl + G

Création d'une fonction numérique complexe d'une variable complexe.

Ctrl + I

Modification d'un objet numérique (icône ).

Ctrl + L

Création d'un affichage de valeur lié à un point?

Ctrl + K

Création d'un commentaire lié à un point.

Ctrl + M

Création d'une marque d'angle non orientée (icône ).

Ctrl + N

Création d'une nouvelle figure vierge avec une longueur unité.

Ctrl + J

Création d'un commentaire libre (icône ).

Ctrl + O

Ouverture d'un fichier (icône ).

Ctrl + P

Impression de la figure (icône ).

Ctrl + Q

Création d'une droite par une équation (icône )

Ctrl + S

Enregistrement de la figure en cours (icône ) .

Ctrl + V

Création d'une variable.

Ctrl + Ins

Création d'un point défini par ses coordonnées dans un repère (icône )

Ctrl + Z

Annulation de la dernière action sur la figure

Ctrl + Y

Annulation de la dernière annulation
