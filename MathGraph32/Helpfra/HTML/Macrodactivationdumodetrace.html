Macro d'activation du mode trace

Les macros ›› Les différents types de macros ››

Ce type de macro sert à activer le mode traces.

Quand le mode Traces est activé, les traces obtenues via le menu Mouvoir - Trace d'objets ou via une macro générant une trace restent permanentes tant qu'on a pas désactivé ce mode ou redimensionné la fenêtre.

Ce type de macro peut être utile au sein d'une macro exécutant une suite de macros.
