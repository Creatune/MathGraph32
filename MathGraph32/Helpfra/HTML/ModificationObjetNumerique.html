Modification d'un objet ››

Pour modifier un objet numérique , cliquez sur l'icône .

Une boîte de dialogue vous présente la liste de tous les objets de type calcul modifiables.

Cliquez sur l'objet choisi puis sur le bouton Modifier (vous pouvez aussi double-cliquer sur l'objet).

Une boîte de dialogue s'ouvre alors pour vous permettre de modifier l'objet.
