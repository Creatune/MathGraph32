Les icônes de la barre supérieure

Les icones ››

Pour créer une nouvelle figure munie d'un longueur unité.

Pour ouvrir une figure à partir d'un fichier.

Pour créer une nouvelle figure munie d'un repère sans vecteurs.

Pour enregistrer la figure en cours d'édition.

Pour imprimer la figure en cours.

Pour annuler la dernière action sur la figure. La barre d'information vous indique quelle était sa nature.

Pour répéter la dernière action annulée sur la figure.

Pour zoomer sur la figure.

Pour dézoomer sur la figure.

Pour réafficher toute la figure en recalculant les tirages aléatoires par la fonction rand() s'il y en a.

Pour translater la figure en faisaint glisser la souris.

Pour capturer un objet : point libre, point lié, marque d'angle, affichage de valeur, commentaire, intitulé de macro.

Pour nommer un point ou une droite. Si Le nom d'un point ou une droite est proche du pointeur de la souris vous pouvez le faire glisser.

Pour changer la couleur ou le tracé d'une objet (style de trait, de point, style de flèche, style de remplissage).

Pour modifier un objet graphique quand celui-ci nécessite une boîte de dialogue pour être changé (F7). Par exemple, cliquer sur un point image d'un autre par une rotation fera apparaître une boîte de dialogue permettant de changer l'angle de la rotation. Cliquer simplement sur l'objet.

Pour afficher les objets numériques, les modifier ou les reclasser (F6).

Pour détruire un objet graphique ainsi que ceux qui en dépendent.

Pour détruire un objet numérique ainsi que ceux qui en dépendent.

Pour masquer un objet graphique (il existe toujours mais n'est plus visible).

Pour démasquer un objet qui a été masqué. Les objets masqués sont tracés en pointillés pour la plupart.

Pour voir l'historique de la figure. (touche de raccourci F5).

Pour connaître la relation entre un point et un objet ou entre deux objets du type droite, demi-droite, segment, cercle ou arc de cercle.

Pour afficher le texte de la figure.

Pour exécuter une macro en cliquant sur son intitulé.

Activation ou non du mode traces pour la figure en cours. Quand ce mode est activé, les traces obtenues à l'aide du menu Mouvoir - Trace d'objets ou à l'aide de macros créant une traces sont conservées. Cet outil fonctionne en bascule.
