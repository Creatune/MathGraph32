Point sur courbe par abscisse

Les courbes et graphes ››

Il ne s'agit pas d'un objet mais d'une construction interne qui vous permet de créer un point sur une courbe de fonction en spécifiant son abscisse.

Vous devez utiliser l'icône ou  le menu Créer - Point - Point sur courbe de fonction - Par abscisse.

Pour cela votre figure doit comprendre un  repère.


Une boîte de dialogue s'ouvre.

Cliquez sur la fonction choisie et le repère.

Entrez l'abscisse du point que vous devez créer. Vous pouvez faire référence à une valeur existante ou entrer un calcul. Dans le deuxième cas, un calcul sera rajouté à la liste des objets céés dont le nom commencera par abs.

Entrez le nom choisi pour le calcul qui contiendra l'ordonnée du point créé (image de l'abscisse).

Validez.

Un point apparaît qui est sur la courbe de fonction. C'est un objet final d'une construction nommée Point sur courbe par abscisse.
