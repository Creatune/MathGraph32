La fonction dérivée partielle

Les objets numériques ›› Les objets numériques de type calcul ››

MathGraph32 peut calculer de façon formelle la dérivée partielle d'une fonction numérique de deux ou trois variables réelles.

Il faut pour cela utiliser le menu Calculs - Calcul dans R - Fonction dérivée partielle.


Une boîte de dialogue s'ouvre vous présentant toutes les fonctions de deux ou trois variables présentes.

Sélectionnez la fonction, précisez le nom de la fonction dérivée et indiquez quelle est la variable par rapport à laquelle il faut dériver.


Remarque : Il s'agit d'un calcul formel mais le calcul de la dérivée partielle n'est pas accessible à l'utilisateur.
