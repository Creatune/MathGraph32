Macro de déplacement de point lié

Les macros ›› Les différents types de macros ››

Une macro de déplacement d'un point lié permet, lorsqu'on l'active, d'obtenir le déplacement d'un point lié à l'emplacement d'un autre point qui est lié au même objet.

On crée un macro de déplacement d'un point lié à l'aide du menu menu Macros >> Nouvelle macro >> Déplaçant un point lié
