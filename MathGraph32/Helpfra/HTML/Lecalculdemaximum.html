Le calcul de maximum

Les objets numériques ›› Les objets numériques de type calcul ››

Il permet de calculer la valeur de x pour laquelle une fonction f est maximale sur un intervalle [a; b].

La valeur n'est assurée d'être correcte que si la fonction est croissante de a jusqu'au maximum et ensuite décroissante jusqu'à b.


Vous devez, dans la boîte de dialogue qui s'ouvre :

  • Choisir la fonction dans la liste des fonctions déjà créées.
  • Choisir la valeur de a.
  • Choisir la valeur de b.
  • Choisir l'incertitude souhaitée.
