Retrait d'objets d'une macro

Les macros ››

Certaines macros gèrent une liste d'objets : les macros d'apparition d'objets, de masquage d'objets, d'animation avec trace, de tace auto par poitn lié ou par variable, de boucle avec animation ou de boucle avec trace.


Pour retirer des objets d'une telle macro après qu'elle ait été créée, utilisez le menu Macro >> Retrait d'objets à une macro.

Une boîte de dialogue s'ouvre vous présentant les macros disponibles.

Cliquez sur la macro de votre choix.

Les objets gérés par la macro clignotent.

Cliquez ensuite sur les objets clignotant que vous voulez retirer de la macro puis cliquez sur le bouton droit quand vous avez fini.

Au moins un objet doit rester dans la macro.
