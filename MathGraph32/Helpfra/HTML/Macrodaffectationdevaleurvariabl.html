Macro d'affectation de valeur à variable

Les macros ›› Les différents types de macros ››

On crée une macro d'affectation de valeur à une variable à l'aide du menu Macros >> Nouvelle macro >> Affectation d'une valeur à une variable.


Une macro d'affectation d'une valeur à une variable sert, lorsqu'on l'active, à  donner une valeur à une variable.

La valeur affectée peut être une constante ou un calcul faisant référence à d'autres valeurs précédemment définies.
